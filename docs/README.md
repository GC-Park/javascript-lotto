# 미션 - 로또 
## 프로젝트 설명
- 로또 번호를 생성해주고 사용자는 구입할 수 있으며 당첨 금액을 확인할 수 있는 프로젝트입니다.

---

## 기능 목록 

1) 기본적인 기능
   - 로또 구입 금액 입력 기능
   - 발행한 로또 수량 및 번호를 출력 기능
   - 로또 번호는 오름차순으로 정렬 기능
   - 당첨 번호를 입력 기능
   - 보너스 번호 입력 기능
   - 당첨 내역을 출력 기능
   - 수익률 구하는 기능


2) 예외처리 기능
    - 로또 당첨 번호의 길이가 6개와 다르면 예외 처리
    - 로또 번호가 1~45 사이의 숫자가 아니면 예외 처리
